PrevisãoDev: 
Previsão do Tempo com IA é um projeto web inovador de previsão do tempo impulsionado por IA. 
Ele fornece dados meteorológicos em tempo real para cidades como Imbituba, com sugestões inteligentes de roupas baseadas na temperatura, umidade e condições climáticas, 
além de suporte a comandos de voz para buscas hands-free. 
  Funcionalidades Principais Busca de previsão do tempo por cidade, exibindo temperatura (ex: 25°C), umidade (ex: 94%) e condições atuais. 
  Sugestões personalizadas de roupas via IA, como "use agasalho leve com protetor solar em dias quentes e úmidos". 
  Comando de voz com reconhecimento de fala por IA para consultas rápidas e intuitivas.
  Interface mobile-friendly com design moderno, incluindo buscas recentes e visual limpo.
  Desenvolvido com tecnologias front-end acessíveis para desenvolvedores em formação:
 
 Tecnologias Utilizados:
  HTML/CSS: Estrutura e design responsivo com fundos dinâmicos.
 
  JavaScript: Integração de APIs de tempo (ex: OpenWeatherMap) e lógica de IA.
  
  IA (Prompt Engineering): Geração de sugestões de roupas e reconhecimento de voz.

  Projeto criado por um dev front-end em ascensão de Imbituba/SC, com certificações em HTML/CSS/JS e fundamentos de IA.
